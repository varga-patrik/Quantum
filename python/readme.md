# MPC320 Controller Project

This project aims to optimally and concurrently set 4 MPC320 polarisers.

## Requirements
- Python 3.10.x
- Thorlabs Kinesis (64-bit)

## Installation
1. Clone this repository.
2. Install dependencies:
   ```sh
   pip install -r python_requirements.txt --only-binary :all:
   ```
3. Download and install Thorlabs Kinesis (64-bit).
